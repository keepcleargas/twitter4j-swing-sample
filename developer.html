How to contribute to the project

Possibilities of Contribution

There are different possibilities for you to contribute to the project:

  • Usage:
    • Write an user manual
    • Translate it into other languages
    • Report bugs
    • Help other users
    • Advertise the project
  • Project Home Page:
    • Design the website for the project
    • translate it into other languages
  • Source Code:
    • Implement new functionalities for the client
    • Fix bugs that are reported
    • Make the client look better! :)

Source Code

You can find, download, and read the source code of the project in the project repository on GitHub.
